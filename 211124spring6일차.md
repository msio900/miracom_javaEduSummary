## 트랜잭션 처리

![](./image/1124-04.png)

```java
@Service
@Transactional
public class MyProductServiceImpl implements MyProductService {
	
	@Autowired
	private MyProductDAO myProductDAO;
	
	@Transactional
	@Override
	public void addProduct(MyProduct vo) throws Exception {
		myProductDAO.addProduct(vo);
	}
```

* 트랜잭션 처리는 Service Layer에서 한다.
